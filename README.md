# InstaGainX

## Project Overview
InstaGainX is a user-friendly web application designed to facilitate logging into Instagram. With a sleek interface, it employs modern styling techniques using Tailwind CSS, providing an engaging experience for users. The app features a simple login form with built-in validation and a loading overlay to enhance user interaction during authentication.

## Installation
To set up InstaGainX locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/InstaGainX.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd InstaGainX
   ```

3. **Open `index.html` in your web browser.** There is no backend server required to run this project, as it solely consists of client-side HTML, CSS, and JavaScript.

## Usage
1. Navigate to the directory where `index.html` is located.
2. Open the `index.html` file in your preferred web browser.
3. Enter your Instagram username and password in the respective fields.
4. Click on the "Log In" button to simulate a login attempt. A loading spinner will indicate the authentication process before redirecting you to the Instagram homepage.

## Features
- Clean and responsive design with Tailwind CSS.
- Input fields for username and password with basic validation.
- Loading overlay to enhance user experience during the authentication process.
- Access to a "Forgot password?" feature link for user convenience.

## Dependencies
The project relies on the following external libraries:
- [Tailwind CSS](https://tailwindcss.com/)
- [Font Awesome](https://fontawesome.com/)

These libraries are included via CDN links in the `index.html` file.

```
InstaGainX/

```

The application is structured to keep all necessary components within a single HTML file, making it easy to navigate and customize.
